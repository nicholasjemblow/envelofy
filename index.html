About Envelofy

Envelofy is a personal finance management tool built around the envelope budgeting method. Categorize your spending, manage accounts, and uncover insights with machine learning analytics. Featuring a glitchy yet innovative LLM assistant (supporting Groq, OpenAI, Ollama, or JLlama), Envelofy offers real-time non-financial advice to help you stay on top of your budget.

Key Features

Envelope Budgeting

Create and manage budget envelopes to track spending by category.

Subscription Finder

Automatically detect and manage recurring subscriptions in your transactions.

Smart Insights

Get AI-driven analytics for unusual spending and budget predictions.

LLM Assistant

A configurable assistant offering non-financial guidance (warning: glitchy!).

Account Management

Monitor all your financial accounts in one unified dashboard.

Data Import

Easily import transactions from CSV files for quick setup.

Get Started

To set up Envelofy, you’ll need JDK 17+, Maven, and a web browser. Follow these steps:

git clone https://github.com/nicholasjemblow/envelofy.git
cd envelofy
mvn clean package
java -jar target/envelofy-1.0.0.jar

Then, visit http://localhost:8081 in your browser.

Configure Your LLM Assistant

Customize the LLM assistant in the app’s Settings screen. Choose from Groq, OpenAI, Ollama, or JLlama, and input your API keys or endpoints as needed.
